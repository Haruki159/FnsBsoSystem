# АИС «Учет БСО»

**АИС «Учет БСО»** — это автоматизированная информационная система, предназначенная для пономерного учета бланков строгой отчетности (БСО). Система автоматизирует ключевые бизнес-процессы административно-хозяйственного отдела, включая приемку, хранение, выдачу и списание документов.

Система обеспечивает централизацию учета и строгий контроль движения каждого экземпляра документа в соответствии с ТЗ.

## 📋 Основные возможности

*   **Учет бланков (Склад):** Просмотр полного списка пачек и отдельных бланков с указанием серии, номера, владельца, остатка и статуса.
*   **Управление движением БСО:**
    *   **Поступление:** Регистрация новых партий бланков (серия, диапазон номеров).
    *   **Выдача:** Оформление передачи бланков от кладовщика сотруднику (МОЛ).
    *   **Списание:** Регистрация уничтожения или порчи бланка (с указанием причины).
    *   **Редактирование:** Изменение статуса, владельца или даты операции.
*   **Аудит и История:** Журнал операций, фиксирующий все действия пользователей (кто, когда и что сделал).
*   **Администрирование:**
    *   Регистрация новых пользователей (ролевая модель).
    *   Личный кабинет пользователя.
*   **Отчетность:** Экспорт текущих остатков и статусов в Microsoft Excel для инвентаризации.
*   **Поиск и Фильтрация:** Быстрый поиск по серии, номеру или году.

## 👤 Роли пользователей

Система ориентирована на следующий персонал:
1.  **Администратор:** Управление учетными записями, доступ ко всем функциям.
2.  **Кладовщик / Менеджмент:** Работа со складом, прием и выдача бланков.
3.  **Специалист АХО:** Оперативная работа с системой.

## 💻 Системные требования

*   **Операционная система:** Windows.
*   **Наличие учетной записи:** Логин и пароль выдаются Администратором.

## 🚀 Запуск и начало работы

1.  Запустите исполняемый файл приложения: `FnsBsoSystem.exe`.
2.  В появившемся окне авторизации введите ваш **Логин** и **Пароль**.
3.  Нажмите кнопку **«Войти»**.
4.  После успешного входа откроется **Главный экран (Landing Page)** с кнопками быстрого перехода:
    *   *«Подробнее»* — переход в Журнал операций.
    *   *«Перейти на склад»* — переход в Книгу учета бланков.

## 🛠 Функциональные модули

### 1. Склад (Книга учета бланков)
Основное рабочее место.
*   **Приход:** Нажмите кнопку «Приход», выберите тип документа, введите серию и диапазон номеров.
*   **Выдача:** Нажмите «Выдача», выберите пачку-источник и получателя (сотрудника).
*   **Списание:** Нажмите «Списание», введите номер бланка и причину (например, «Технический брак»). *Внимание: действие необратимо!*
*   **Экспорт:** Нажмите «Excel» для выгрузки отчета.

### 2. Журнал операций (История)
Обеспечивает функцию аудита. Таблица отображает дату, время, пользователя, тип действия и подробности операции.

### 3. Личный кабинет и Регистрация
*   **Аккаунт:** Просмотр личных данных и смена пароля.
*   **Регистрация (только для Админа):** Создание новых пользователей с заполнением ФИО, отдела, должности и назначением роли (например, Viewer, Admin).

## 📖 Инструкция пользователя

Ниже описан порядок работы с основными модулями системы.

### 1. Авторизация
1. При запуске `FnsBsoSystem.exe` откроется окно входа.
2. Введите **Логин** и **Пароль**, полученные от Администратора.
3. Нажмите кнопку **«Войти»**.

### 2. Работа со складом (Книга учета)
Перейдите в модуль, нажав кнопку **«Склад»** в верхнем меню или «Перейти на склад» на главном экране. Здесь отображается таблица всех бланков.

#### 2.1. Поступление новых бланков (Приход)
Используется при регистрации новой партии бланков, поступивших в ИФНС.
1. Нажмите кнопку **«+ Приход»**.
2. В открывшемся окне выберите **Тип документа** из выпадающего списка.
3. Укажите **Серию** документа.
4. Введите **Диапазон номеров** (Начало и Конец).
5. Нажмите **«Зарегистрировать приход»**.

#### 2.2. Выдача сотруднику
Используется для оформления передачи бланков от Кладовщика материально ответственному лицу (МОЛ).
1. Нажмите кнопку **«Выдача»**.
2. В поле **Источник** выберите пачку, из которой будут выданы бланки.
3. В поле **Получатель** выберите сотрудника.
4. Укажите **Количество** выдаваемых штук.
5. Нажмите **«Оформить выдачу»**.

#### 2.3. Списание (Уничтожение)
Используется при порче бланка или техническом браке.
1. Нажмите кнопку **«Списание»**.
2. Введите конкретный **Номер** испорченного бланка.
3. Выберите **Причину списания** (например, «Технический брак»).
4. Нажмите красную кнопку **«УНИЧТОЖИТЬ БЛАНК»**.
> **Важно:** Система выдаст предупреждение. Это действие необратимо.

#### 2.4. Редактирование и Поиск
*   **Редактирование:** Выделите строку в таблице и нажмите **«Ред»**. Можно изменить статус (например, "В подотчете"), владельца (МОЛ) или скорректировать количество.
*   **Поиск:** Используйте строку поиска над таблицей для фильтрации по серии, номеру или году.

### 3. Экспорт отчетов
Для проведения инвентаризации:
1. Находясь в модуле «Склад», нажмите кнопку **«Excel»** (иконка зеленого цвета).
2. Система сформирует и откроет файл `.xlsx` с текущими остатками и статусами всех бланков.

### 4. Администрирование (Для роли Admin)
#### Регистрация нового пользователя
1. Нажмите кнопку **«Регистрация»** в верхнем меню.
2. Заполните блок **Личные данные**: ФИО, Отдел, Должность.
3. Заполните блок **Доступ**: Придумайте Логин, Пароль и выберите **Роль пользователя** (например, Viewer или Admin).
4. Нажмите «Зарегистрировать сотрудника».
